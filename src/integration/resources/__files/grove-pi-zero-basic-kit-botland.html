Opis produktu: GrovePi Zero Basic Kit - zestaw dla Raspberry Pi Zero - dla początkujących

Zestaw składający się z modułów z serii Grove, umożliwiający tworzenie prostych projektów przez początkujących użytkowników Raspberry Pi Zero. Zawarte w zestawie elementy takie jak np. buzzer czy przycisk to elementy podstawowe, spełniające zazwyczaj jedną, określoną funkcję. Cały system Grove natomiast posiada znacznie więcej dodatków i elementów, dzięki którym można tworzony projekt znacznie bardziej rozbudować o np. akcelerometry, barometry, czujniki gazu, płomieni.

  

Uwaga!

Zestaw nie zawiera Raspberry Pi. Można je nabyć osobno w naszym sklepie.

Dlaczego wybrać moduły z systemu Grove?

  • Moduły Grove posiadają standaryzowane złącza - dzięki temu ich podłączenie nie wymaga lutowania czy spinania przewodów na płytce stykowej.
  • Seria Grove jest kompatybilna z popularnymi minikomputerami:
    • Arduino - np. do płytki głównej Arduino Uno czy Arduino Leonardo możne je podłączyć za pomocą przewodów poprzez nakładkę bazową Grove Base Shield
    • Raspberry Pi - moduły podłączone poprzez nakładkę typu hat Grove Pi
    • BBC micro:bit - poprzez dedykowaną nakładkę Grove dla micro:bit
  • Wszystkie moduły i czujniki zawierają w zestawie przewody do podłączenia z wyżej wymienionymi nakładkami, oprócz tego dostępne są zestawy w przewodów o innych długościach oraz rozmaite przejściówki.
  • System Grove jest rozbudowany, można w nim znaleźć moduły oparte na popularnych czujnikach, sterowniki silników oraz same silniki i aktuatory np.:
    • akcelerometry i żyroskopy, przekaźniki
    • silniki i serwa oraz sterowniki
    • czujniki: w tym m.in.: pogodowe, medyczne, jakości powietrza, światła
    • enkodery, potencjometry, przyciski i joysticki
    • moduły dźwiękowe i komunikacyjne
    • diody LED i wyświetlacze
  • Zestawy startowe Grove, dostępne w wersji dla Raspberry Pi, Arduino i micro:bit ułatwiają wybór początkującym elektronikom.
